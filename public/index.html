사용 방법 가이드

1. 수정할 링크 입력 후 다운로드

수정할 링크를 입력한 후 '다운로드' 버튼을 누릅니다

2. 파일 삭제 및 업로드

압축 해제된 파일에서 X 버튼을 눌러서 삭제하거나, + 버튼을 눌러서 새 파일을 업로드합니다.

3. 재압축

'재압축하기' 버튼을 누릅니다

4. 서버 업로드

'윈도우 서버에 업로드' 버튼을 누릅니다

1. 서버 파일 조회

'서버 파일 조회 / 삭제 / 업로드' 버튼을 누릅니다

2. 기존 파일 관리

서버에 있는 기존 파일들을 확인하고, 필요하면 X 버튼으로 삭제할 수 있습니다.

3. 새 파일 선택

업로드할 파일을 선택하고 서버 파일명을 입력합니다.

💡 중요: 서버에 이미 존재하는 파일명과 동일한 이름으로 업로드하면, 기존 파일이 새 파일로 교체됩니다. 이렇게 하면 기존 다운로드 링크는 그대로 유지되면서 파일 내용만 새로운 것으로 바뀝니다.

4. 파일 업로드

'파일 업로드' 버튼을 눌러 서버에 업로드합니다
